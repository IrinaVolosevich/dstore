# Удаление сортировки заведения в каталоге
1. Достать документ сортировки, вместе с его связью с рубрикой (ссылками на эти объекты)
2. Удалить данный документ у заведения
3. Удалить документ сортировки каталога

# Удаление рубрики
1. Найти все заведения связанные с этой рубрикой
2. Найти все сортировки связанные с этими рубриками и заведениями
3. Удалить сортировки у соответствующих заведений
4. Удалить сортировоки
5. удалить рубрику

# Удаление заведения
1. Найти все сортировки связанные с этим заведением
2. Удалить сортировки
3. Найти адрес
4. Удалить адрес
5. Найти услуги заведения
6. Удалить услуги
7. Удалить заведение
